# AutoPortal 功能特性

## 主要功能

- ✅ **自动登录** - 连接校园网时自动检测并登录
- ✅ **记住密码** - 安全保存账号密码
- ✅ **状态检测** - 实时检测网络和登录状态
- ✅ **主题切换** - 支持深色/浅色主题
- ✅ **Mica 背景** - 现代化毛玻璃效果
- ✅ **简洁界面** - 现代化 UI 设计，操作简单
- ✅ **卡片管理** - 首页支持卡片拖拽、上移/下移、添加/删除
- ✅ **布局保存** - 自动保存卡片顺序和位置

## 卡片管理

首页支持灵活的卡片布局管理：

### 右键菜单

在任意卡片上点击鼠标右键，可以：
- **添加新卡片** - 在当前卡片后添加指定类型的新卡片
- **上移** - 将卡片向上移动一个位置
- **下移** - 将卡片向下移动一个位置
- **删除此卡片** - 删除当前卡片（状态卡片和使用说明卡片除外）
- **重置所有卡片位置** - 恢复默认布局

### 视觉反馈

- 右键点击卡片时，卡片会显示蓝色边框高亮
- 菜单在鼠标指针位置显示
- 菜单关闭后自动恢复原状

### 布局保存

- 卡片顺序会自动保存到配置文件
- 重启应用后自动恢复上次的布局
- 配置文件位置：`%LOCALAPPDATA%\AutoPortal\cardOrder.json`

### 支持的卡片类型

- 流量统计卡片
- 连通测试卡片
- IP 信息卡片
- 系统信息卡片
- 使用说明卡片
